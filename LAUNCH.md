## Short Post

I made Website Terminal: a safe browser-only terminal widget for any static site.

You add commands and answers, preview the terminal, then copy the embed code. No dependencies, no build step, no real shell execution.

Demo: https://mir4zzz.github.io/website-terminal/
Repo: https://github.com/MIR4ZZZ/website-terminal

## Reddit / Hacker News

Show HN: Website Terminal - a copy-paste fake terminal builder for normal websites

I built a tiny browser-only terminal widget for static sites, portfolios, docs, and landing pages. It has a builder UI where you add commands and answers, preview the result, and copy the embed code.

It is plain HTML/CSS/JS, has no dependencies, and does not execute real commands.
